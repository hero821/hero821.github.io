安装

  1. 【所有节点】启用2个网卡，ens192需要配置IP，ens224无需配置IP

    略

  2. 【所有节点】关闭 selinux、firewalld、NetworkManager，开启 ip_forward

    略

  3. 【所有节点】修改 yum源、pip源

    略

  4. 【所有节点】修改 hostname、hosts

  5. 【所有节点】安装 python 虚拟环境

    推荐，能少遇到很多 python 版本兼容的问题，结合 kolla-ansible 的 ansible_python_interpreter 一起使用

  6. 【所有节点】安装 docker

  7. 【所有节点】给硬盘添加分区标签

  8. 【仅deploy节点】配置免密登录

  9. 【仅deploy节点】安装 ansible、kolla、kolla-ansible
